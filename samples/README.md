# Talkment 사내용 스프레드시트 샘플

사내 임직원 자산·소모품 관리용 구글 스프레드시트 템플릿입니다.

## 필수 탭 (5개)

탭 이름을 **아래와 완전히 동일**하게 만드세요.

```
명단
시리얼 물품 관리
시리얼 입출고 내역
일반 물품 관리
일반 입출고 내역
```

## 탭 구조

| 탭 | 용도 |
|----|------|
| **명단** | 임직원 |
| **시리얼 물품 관리** | 시리얼 물품 마스터 (항목·시리얼·상태) |
| **시리얼 입출고 내역** | 시리얼 출고·반납 이력 |
| **일반 물품 관리** | 소모품 재고 마스터 |
| **일반 입출고 내역** | 소모품 출고 기록 |

### 1행 헤더

**명단:** `No, 이름, 직함, 소속, 연락처, 이메일주소`

**시리얼 물품 관리:** `No, 항목, 시리얼 넘버, 상태`

**시리얼 입출고 내역:** `No, 항목, 시리얼 넘버, 출고일, 출고자, 직함, 소속, 연락처, 이메일, 반출일, 반납일, 비고, 출고서명, 반납서명, 사진`

**일반 물품 관리:** `No, 항목, 초기 재고수량, 출고 총갯수, 현재 잔여갯수, 비고`

**일반 입출고 내역:** `No, 항목, 출고갯수, 출고일, 출고자, 직책, 소속, 연락처, 이메일, 서명`

## 샘플 파일

| 파일 | 설명 |
|------|------|
| `Talkment_사내용_샘플.xlsx` | 5탭 + 예시 데이터 |
| `사내용_csv/` | 탭별 CSV |

## 등록 방법

1. Google Drive에서 스프레드시트 생성 또는 xlsx 가져오기
2. 탭 이름·헤더 확인
3. 서비스 계정에 **편집자** 공유
4. Talkment 앱에서 URL 등록 → **연결 테스트**

## 샘플 재생성

```bash
cd samples
node generate-samples.mjs
```

## 앱 동작 요약

- **시리얼 출고:** 물품 관리 탭 `상태` → `반출`, 입출고 관리 탭에 이력 추가
- **시리얼 반납:** 입출고 관리 탭 `반납일` 기록, 물품 관리 `상태` 비우기
- **소모품 출고:** 입출고 내역 추가 + 물품 관리 재고 갱신
